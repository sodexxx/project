# Учебный проект на языке C: Модульная структура

Проект представляет собой сборник из трех лабораторных работ, объединенных в единую программу с консольным меню. Реализован модульный подход с разделением на заголовочные (.h) и исходные (.c) файлы.

## Структура проекта

- `include/` — заголовочные файлы с прототипами функций.
- `src/` — реализация функций лабораторных работ.
- `main.c` — главный файл с меню выбора задач.

## Лабораторные работы

1. Лабораторная работа №1. Линейные алгоритмы
2. Лабораторная работа №2. Циклические алгоритмы
3. Лабораторная работа №3. Обработка потока символов

## Как собрать и запустить

Для компиляции проекта требуется компилятор GCC.
